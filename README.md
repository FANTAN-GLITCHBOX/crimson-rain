# Crimson Rain

A browser-based vertical scrolling bullet-hell shooter.

## Play

Open `index.html` in a browser, or publish this repository with GitHub Pages.

## Controls

- Arrow keys / WASD: Move
- Z / Space: Shot, start, restart
- X / C: Bomb
- Shift: Slow move
- P: Pause
- Gamepad: Left stick / D-pad to move, face button to shoot, another face button to bomb
- Touch: Drag to move, double-tap to bomb

## Debug Mode

Type `debug` on the game screen to show the debug panel.

- Stage Start
- Boss Start
- Power + / Power -
- Medal + / Medal -
- Wasp Boss
- Invincible toggle

## Notes

High scores are saved in the browser's LocalStorage.
